# 小新书-一个分享和探索编程知识的地方。

目前包含：

- 刷题平台

- API 接口开放平台
- 协同云图库



## 刷题平台

### 项目介绍

基于**Spring Boot + MySQL + Redis + Elasticsearch + React + Ant Design Pro** 的 **面试刷题平台**

管理员可以创建题库并批量关联题目；用户可以分词检索题目、在线刷题并查看刷题记录日历等



## API 接口开放平台

### 项目介绍

提供 API 接口供开发者调用的平台，基于 Spring Boot + React 的全栈微服务项目

用户通过注册登录，开通接口调用权限，浏览接口并进行调用，且每次调用都会进行统计，还可以通过使用 客户端 SDK 在自己的项目中调用接口。

管理员可以发布接口、下线接口、接入接口、以及可视化接口的调用情况和数据
